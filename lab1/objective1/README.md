# 實驗目標一  
透過Modelsim進行模擬輸入輸出W、X、Y、Z、a、b、c、d、e、 f和g,在燒入開發板用指撥開關(4個Switch)控制七段顯示器顯 示0 ~ F之間任意數字。

## 指定腳位  
- 下表為範例所使用七段 顯示器為HEX0,如需使 用到其他顆七段顯示器 ,請自行查詢規格書第 36 、37 、38頁。 (Lecture0可供下載)。
- 完成之後按下實驗板上自己設定之腳位的指撥開關,即可輸入4-bit訊號給七段顯示器。
